# AgentMed - Улучшенный AI агент

Улучшенная версия AI агента с самомодификацией.

## Новые возможности:
- Планировщик задач
- Сохранение истории чатов
- Пользовательские данные
- Автоматическое применение изменений кода

## Использование:
```bash
python agent_med.py
```

## Файлы:
- `agent_med.py` - основной код
- `user_data.json` - пользовательские данные
- `chat_history.pkl` - история чатов
- `agent_med.log` - логи
