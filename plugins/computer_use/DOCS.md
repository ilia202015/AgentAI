# Computer Use Plugin Documentation

Этот плагин позволяет AI-агенту взаимодействовать с графическим интерфейсом операционной системы.

## Инструменты

### 1. computer_screenshot
**Описание**: Делает снимок всего экрана.
**Возвращает**: Текстовое описание (разрешение) и изображение в формате base64.
**Когда использовать**: Как первый шаг для оценки ситуации на экране.

### 2. computer_analyze_region
**Параметры**: `x1, y1, x2, y2` (координаты прямоугольника).
**Описание**: Вырезает часть экрана, распознает текст (OCR) и ищет UI-элементы.
**Возвращает**: Список найденных элементов с их ГЛОБАЛЬНЫМИ координатами.
**Когда использовать**: Для точного определения координат кнопок, полей ввода или чтения текста в конкретной области.

### 3. computer_actions
**Параметры**: `actions` (JSON-массив объектов).
**Описание**: Выполняет серию низкоуровневых действий (клик, ввод, скролл, горячие клавиши).
**Примеры действий**:
- `{"action": "click_at", "x": 100, "y": 200}`
- `{"action": "type_text_at", "x": 100, "y": 200, "text": "Hello", "press_enter": true}`
- `{"action": "key_combination", "keys": "ctrl+c"}`
**Важно**: Все координаты передаются в абсолютных пикселях.

## Архитектура
Плагин интегрируется напрямую в основной цикл агента. Модель видит эти инструменты в общем списке и может комбинировать их с поиском, работой с файлами и браузером.
