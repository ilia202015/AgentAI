Центральный узел

Класс Chat — это ядро агента. Он управляет API нейросети, хранит историю контекста, предоставляет доступ к инструментам и поддерживает локальное окружение (self.local_env).

Базовый инструментарий ОС

Инструмент Python

Исполнение сгенерированного кода напрямую в локальном окружении процесса.

Инструмент Shell

Интерактивная сессия терминала с сохранением контекста между вызовами.
