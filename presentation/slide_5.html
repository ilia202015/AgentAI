Вспомогательные Инструменты Ядра

python_str

Специализированный инструмент форматирования. Принимает текст и возвращает его как Python-строку, экранируя спецсимволы. Критически важен для безопасной записи кода в файлы.

google_search

Интеграция с Google Custom Search API. Позволяет агенту самостоятельно искать актуальную информацию в интернете, компенсируя устаревание весов модели.

sandbox

Управление локальной песочницей. Копирует проект во временную папку и запускает изолированный процесс сервера. Позволяет агенту безопасно тестировать свои изменения.

http

Загружает веб-страницы по URL, очищает HTML от тегов и скриптов с помощью BeautifulSoup, возвращая агенту чистый, легко анализируемый текстовый контент.

user_profile

Система долгосрочной памяти. Сохраняет в JSON личные предпочтения пользователя, профессиональный контекст и общие инструкции, обеспечивая непрерывность между сессиями.
